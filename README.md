Accessing the Belgian Identity Card from VB.NET
===============================================

Visual Basic code on how to use the Belgian eID-middleware to read the eID-smartcards.

You can find more information about the code on my blog.

* [20/01/2008: Accessing the Belgian Identity Card from VB.NET](http://cgeers.com/2008/01/20/accessing-the-belgian-identity-card-from-vbnet-part-1/)

The blog article is also available as a PDF document in the doc folder.
